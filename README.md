# Expense Analyzer (Spring Boot Project)

## 📌 Description
A full-stack backend project to manage personal expenses and income.  
Users can register, login, and track their financial transactions.

## 🚀 Features
- User Registration & Login
- Add Expense / Income
- View All Transactions
- MySQL Database Integration

## 🛠️ Tech Stack
- Java (Spring Boot)
- MySQL
- Spring Data JPA
- REST APIs

## 📊 API Endpoints
### Auth
- POST /api/auth/register
- POST /api/auth/login

### Transactions
- POST /api/transactions
- GET /api/transactions

## ▶️ How to Run
1. Clone the repo
2. Configure MySQL in application.properties
3. Run the Spring Boot application
4. Test APIs using Postman
